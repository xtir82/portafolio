Lenguajes que uso actualmente..

HTML

HTML es el lenguaje de marcado utilizado para crear y diseñar documentos en la World Wide Web. HTML es el lenguaje utilizado en este portafolio y otros proyectos para la creación de la estructura de cada página.

CSS

CSS es un lenguaje de estilos. CSS es el lenguaje utilizado para definir como se muestran los elementos en nuestro sitio web. Colores, fuentes, espaciado y otros aspectos visuales. En este portafolio utilizo el framework Bootstrap y el preprocesador SASS, los cuales son elementos que mejoran la experiencia de CSS.

JavaScript

JavaScript es un lenguaje de programación interpretado de alto nivel que se utiliza principalmente para crear contenido dinámico e interactivo en sitios web. Para los efectos de animación y algunas funcionalidades del sitio se ha utilizado JS. ¡Pronto tendré más proyectos en JS en la sección de Portafolio!
